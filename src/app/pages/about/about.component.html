About

¿Qué es Jobtion?

Jobtion es la aplicación para contratar trabajadores independientes mediante una interacción sencilla con tarjetas. Haciendolo facil para toda aquel que quiera contratar personas en cualquier categoria de trabajo.

¿En qué está hecha Jobtion?

Jobtion está hecha con Ionic, este es un SDK completo de código abierto para el desarrollo de aplicaciones móviles híbridas.

Para la parte del backend Jobtion está programada con Spring/Springboot, un framework para el desarrollo de aplicaciones y contenedor de inversión de control, de código abierto para la plataforma Java.​
